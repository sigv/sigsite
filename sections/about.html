sigsite is a static website delivery framework.

It takes in a bunch of files you provide.

It also takes in HTTP requests browsers send.

It spits out data for the browsers to handle.

It uses fairly well estabilshed technologies.

Node.js powers the core, Express helps out with requests.

Additional magic has been introduced for increased awesomeness.

No rocket science involved, you know.

You build your assets, we provide them as efficiently as possible.

The Github project page has more details on setup.

Simplicity re-imagined.
